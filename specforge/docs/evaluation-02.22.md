What Is It About                                                                                                            
                                                                                                                                
 SpecForge is a CLI tool (with optional Web UI) that takes a Markdown product specification as input and automatically          
 generates a complete, tested, Docker-ready FastAPI microservice as output. It uses a multi-agent AI pipeline to do this.       
                                                                                                                                
 Think of it as: "Write a requirements doc in Markdown â†’ get a fully working API with tests, Docker, and docs."
